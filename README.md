# MMED-1055-HW2-Crushing Bugs
This is the repo for HW2 Crushing Bugs in MMED 1055.

## Features
- Drag and drop labels onto target zones
- Reset button to restart the game
- Visual feedback when hovering over drop zones

## Bug Fixes Implemented
1. **Multiple Labels in Drop Zone**  
   - Fixed: Now prevents multiple labels from being dropped in the same zone

2. **Reset Functionality**  
   - Fixed: Reset button now properly returns all labels to their original container

3. **Bonus: Drop Zone Highlighting**  
   - Added: Visual feedback when dragging over target zones

## How to Play
1. Drag the organ labels from the right side
2. Drop them onto the correct body parts on the diagram
3. Click "Reset Game" to start over

## Branches
1. `bug.cy.multi-drop-fix` - Fix for multiple labels in drop zones
